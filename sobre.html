Sobre o Projeto

EngPro

Portal técnico de referência em Engenharia de Produção. Conhecimento rigoroso, organizado e gratuito.

Missão

Democratizar o acesso ao conhecimento técnico de Engenharia de Produção em português, oferecendo conteúdo organizado, profundo e referenciado para estudantes, profissionais e gestores industriais.

O Que Oferecemos

  • 30+ Áreas técnicas completas com centenas de tópicos
  • 500+ Sub-tópicos detalhados com exemplos práticos
  • 8 Modelos CAD gratuitos para download
  • 12 Cursos gratuitos de plataformas reconhecidas
  • 12 Formulários técnicos .doc para download
  • Grátis Conteúdo 100% gratuito e aberto

Modelo de Conteúdo

Cada tópico do EngPro segue um padrão de qualidade: definição clara, contextualização histórica/normativa, aplicação prática com exemplos, dados quantitativos quando aplicável e referências bibliográficas. Todo o conteúdo é revisado periodicamente.

Tecnologia

O site foi construído com HTML5, CSS3 (Tailwind utility classes), JavaScript vanilla, Three.js para animações 3D e GSAP para animações scroll. O deploy é automatizado via GitHub Pages com workflow de CI/CD.

Contribua

O EngPro é um projeto aberto. Você pode contribuir com novos tópicos, correções, modelos CAD, cursos ou formulários. Acesse nosso repositório no GitHub e abra uma issue ou pull request.
